This Flappy Bird clone, developed in C using the SDL2 and SDL_ttf libraries, implements a complete 2D game engine featuring a gravity-driven physics system, AABB collision detection, and procedural object pooling. The architecture utilizes a central Game Loop that handles real-time user input (Spacebar, Mouse, or Touch) to apply upward velocity to a Bird structure, which is constantly pulled down by a gravitational constant. The environmental challenge is created by an array of Pipe structures that move horizontally at a fixed speed; as each pipe exits the screen, it is "recycled" to the right with a randomized height to ensure infinite gameplay without memory leaks.  The system manages game states by detecting intersections between the bird’s bounding box and the pipe segments or screen boundaries, triggering a reset logic that restores the score and positions. Visually, the program demonstrates graphical layering by rendering a sky-blue background, static primitive shapes for clouds and the sun, and dynamic text overlays for real-time score tracking via the TrueType Font (TTF) subsystem.
